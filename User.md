## User experienceの深堀
```
profileの入力
どんなことをしたいかをAIに振る舞いを指定することで作成する。
```

ex)
input:
```
profile: ユーザー像 シンゴさん： シンゴは45歳のフリーランスのTVプロデューサーで、妻のマイコと2人の子供(高校生と中学生)と一緒に神奈川県藤沢市に住んでいます。一家は最近、ココアという名のゴールデンレトリバーの子犬を迎え入れました。 一家は子供たちの部活動遠征のために定期的に旅行に出かけ、シンゴは撮影の際に長時間移動することが多いです。シンゴは、家族が留守にしている間や忙しい時に、子犬のココアの面倒を見てくれる人を探したいと思っています。 シンゴさんが一番心配しているのは、自分の家に来て子犬の世話をしてくれる見知らぬ人を信用することです。シンゴさんは、子犬の適切な世話の仕方を知っていて、ココアを散歩やドッグパークに連れて行ける人を雇うのが一番安心だと考えています。
\ あなたはプロの編集者です。以下の文章を、ユーザープロフィール表の形に要約、成形してください。文章のニュアンスは可能な限り残してください。
```

output:
```
名前: シンゴ
年齢: 45歳
職業: フリーランスのTVプロデューサー
家族構成: 妻と2人の子供(高校生と中学生)、ゴールデンレトリバーの子犬のココア
住所: 神奈川県藤沢市
旅行: 子供たちの部活動遠征で定期的に旅行に出かける
懸念: 見知らぬ人が自宅に入って子犬の世話をすることに不安がある
要望: 子犬の適切な世話をする信頼できる人を探したい
方法: 友人や知人からの紹介、オンラインのペットシッター検索サイトの利用、または地元のペットショップや動物病院での紹介を検討
優先事項: 子犬の安全と健康を確保すること
期待すること: ペットシッターにココアの散歩やドッグパークへの連れ出し、適切な食事や水分補給、トレーニングや社交性向上の促進を期待
```

input:
```
あなたはプロのUXデザイナーです。今度ペットの出張ケアサービスをリリースします。以下のユーザーが現在抱えている問題について箇条書きにしてください。またどんな課題をかかえているでしょうか。類推してください。
```

output:
```
問題点:

子犬の適切な世話をするために信頼できるペットシッターを見つけることが難しい
家に見知らぬ人を入れることに不安がある
仕事で長時間移動するため、子犬の世話をする時間が限られている
子供たちの部活動遠征で定期的に旅行に出かけるため、ペットケアが必要
類推される課題:

シンゴさんは家族の安全や子犬の健康を非常に重視していることがわかるため、ペットシッターの信頼性が非常に重要であることが予想される。
長時間移動するため、柔軟なサービスが必要であることが予想される。
定期的な旅行があるため、サービスの予約や手配が容易であることが望まれる。
```
